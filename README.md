## 🏥 Descripción

Sistema de gestión hospitalaria desarrollado en **Java**, diseñado para optimizar la administración de hospitales, clínicas y centros médicos.  
Permite gestionar pacientes, personal médico, citas, historiales clínicos y procesos administrativos desde una plataforma centralizada.

---

## 🚀 Características Principales

### 👨‍⚕️ Gestión de Pacientes
- Registro de pacientes
- Historial clínico digital
- Control de admisión y alta
- Seguimiento de tratamientos

### 🗓️ Gestión de Citas
- Agenda médica
- Programación de consultas
- Control de disponibilidad de doctores

### 🧪 Área Médica
- Diagnósticos clínicos
- Registro de signos vitales
- Prescripciones médicas
- Resultados de laboratorio

### 🏥 Operaciones Hospitalarias
- Gestión de pacientes internos y externos
- Control de habitaciones
- Registro de cirugías
- Historial médico completo

### 💼 Administración
- Gestión de personal
- Control de inventario médico
- Reportes administrativos
- Gestión de pagos

---

## 🛠️ Tecnologías Utilizadas

- Java
- Swing / JavaFX (según implementación)
- MySQL / PostgreSQL
